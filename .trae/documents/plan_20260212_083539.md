I will implement a dynamic progress bar in the macOS Menu Bar (Tray) icon to visualize daily worklog progress.

### Plan

1. **Frontend Update (`apps/tauri/src`)**:

   * Modify `App.tsx` to fetch daily worklogs using `useWorklogs`.

   * Calculate the daily progress percentage based on an 8-hour workday target.

   * Update `useTrayTimer` hook to accept this `progress` value.

   * Pass the `progress` value to the Rust backend command.

2. **Backend Update (`apps/tauri/src-tauri/src/lib.rs`)**:

   * Rename/Update the `update_tray_title` command to `update_tray_state` (accepting `title` and `progress`).

   * Implement a lightweight icon generator in Rust (using raw byte manipulation, no new heavy dependencies).

   * The icon will be a **Template Image** (adapts to Light/Dark mode automatically):

     * A horizontal progress bar container.

     * Filled portion representing the daily progress.

   * Update the Tray Icon dynamically whenever the timer/progress changes.

### Outcome

* **Tray Icon**: Displays a visual progress bar (e.g., a horizontal bar or box) indicating hours worked vs. 8h goal.

* **Tray Text**: Continues to display the active timer (e.g., "00:05:23 - KEY-123") or "No timer" as requested.

* **Visuals**: Native macOS look (adapts to menu bar color).
